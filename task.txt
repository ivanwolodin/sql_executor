# Задание:
* Написать программу для выполнения SQL запросов и отображения результата
* Python3, PyQt5, DB-API 2.0 (не QtSql!)
* Поле для ввода sqlite3 connection string (default value - :memory:)
* Поле для ввода запроса
* Кнопочка для запуска
* QTableView для отображения результата (не QTableWidget!)
* Вы можете делать разумные предположения о требованиях, не описанных выше. 
 Если у вас возникают вопросы о разумности или значимости какого-то предположения, и вообще какие-либо вопросы - они *приветствуются** в этом чате и не могут повлиять на оценку результата  отрицательно.

## Дополнительно
* Доработать connection string, чтобы могло подключаться к другим БД

## Критерии оценки
* Качество кода
* User-friendliness
* Время выполнения
по опыту предыдущих кандидатов: порядок критериев ВАЖЕН
